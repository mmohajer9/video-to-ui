## [0.1.0] - 2026-05-02

### Added

- Initial public release.
- Five modes: extract frames, analyze design, compare-and-edit, HTML/CSS/JS mockup, Vite + React + TypeScript + Tailwind app.
- 2-tier subagent pattern for context-efficient frame analysis.
- ffmpeg-based frame extraction with duration-aware default fps and a frame budget.
- Claude Code plugin marketplace metadata for one-line install via `/plugin marketplace add`.
